Sophos XGS — DHCP Source
Syslog DHCP Events

Gwless can listen for UDP syslog messages from Sophos SFOS to track DHCP leases in real time — no SSH admin access required. When enabled and receiving events, syslog is preferred over SSH.

—
How to configure Sophos SFOS
  1. In Sophos SFOS, go to Logging & Monitoring → Log Settings
  2. Under Syslog Servers, add a new server:
    IP = this container's IP address, Port = 514, Protocol = UDP, Format = Default (or BSD)
  3. Under Log Type and Severity, ensure Event logs are enabled
  4. Make sure DHCP is included in the event log categories
  5. Click Apply — lease events will appear in the Status field above after the next DHCP activity

Note: Port 514 requires the container to run as root (default for the LXC installer). Use a port > 1024 (e.g. 5140) if running unprivileged.
